# 5S Learning Second Brain — Power BI

Sistema inteligente de aprendizagem utilizando NotebookLM,
Inteligência Artificial,
metodologia 5S
e Gestão à Vista
para apoiar estudantes em diferentes áreas de conhecimento.

---

# Objetivo

Este projeto foi desenvolvido como parte do desafio da DIO
com foco na utilização da Inteligência Artificial
como ferramenta de aprendizagem ativa.

O objetivo principal foi criar um “Segundo Cérebro”
capaz de:
- organizar conhecimento;
- estruturar estudos;
- apoiar revisões;
- melhorar produtividade;
- facilitar aprendizagem contínua.

Como aplicação prática,
foi utilizado o tema Power BI.

---

# Conceito do Projeto

O projeto propõe a criação de um sistema inteligente
de estudos utilizando:
- NotebookLM;
- IA aplicada;
- metodologia 5S;
- Gestão à Vista;
- engenharia de prompts;
- organização do conhecimento.

A proposta é transformar a IA
em uma ferramenta estratégica de aprendizagem.

---

# Aplicação da Metodologia 5S

## Utilização
Separação de conteúdos úteis
e eliminação de informações redundantes.

## Organização
Estruturação de:
- fontes;
- prompts;
- resumos;
- documentação;
- materiais de estudo.

## Limpeza
Revisão contínua de:
- resumos;
- prompts;
- conteúdos;
- materiais documentados.

## Padronização
Padronização de:
- arquivos;
- organização visual;
- documentação;
- prompts reutilizáveis.

## Disciplina
Construção de rotina consistente
de aprendizagem e melhoria contínua.
